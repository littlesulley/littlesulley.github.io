An enhanced Cauchy-Schwarz inquality of the integral form

We all know the very famous Cauchy-Schwaz inequality (of the integral form):

Let fff and ggg be real functions continuous in [a,b][a,b][a,b]. Then the following equality holds:

(∫abf(x)g(x)dx)2≤∫ab(f(x))2dx∫ab(g(x))2dx\left(\int_a^bf(x)g(x)\mathrm{d}x\right)^2\le\int_a^b(f(x))^2\mathrm{d}x\int_a^b(g(x))^2\mathrm{d}x (∫ab​f(x)g(x)dx)2≤∫ab​(f(x))2dx∫ab​(g(x))2dx

The purpose of this blog is to prove an enhanced version of this inequality, which is stated as follows:

Let f,g∈L2[a,b]f,g\in L^2[a,b]f,g∈L2[a,b], 0<m≤f≤M0<m\le f\le M0<m≤f≤M, and ∫abg(x)dx=0\int_a^bg(x)\mathrm{d}x=0∫ab​g(x)dx=0. Prove that

(∫abf(x)g(x)dx)2≤(M−mM+m)2∫ab(f(x))2dx∫ab(g(x))2dx\left(\int_a^bf(x)g(x)\mathrm{d}x\right)^2\le\left(\frac{M-m}{M+m}\right)^2\int_a^b(f(x))^2\mathrm{d}x\int_a^b(g(x))^2\mathrm{d}x (∫ab​f(x)g(x)dx)2≤(M+mM−m​)2∫ab​(f(x))2dx∫ab​(g(x))2dx

Step 1: Substract kkk from fff

From the vanilla C-S inequality, we know that if fff and ggg are linear, i.e., f=kgf=kgf=kg, the equality holds. However in this problem, we know that there does not exist such kkk that f=kgf=kgf=kg because f>0f>0f>0 and ggg has non-positive values in [a,b][a,b][a,b]. This suggests us that, to reach its minimum value as much as possible, we should manage to push fff to be as linear as ggg as possible. To this end, we can substract a constant kkk, from fff, so that fff and ggg can hold the same sign at any given input x∈[a,b]x\in[a,b]x∈[a,b].

So what should we choose for kkk. An intuitive choice is the average of fff, i.e., k=1b−a∫abf(x)dxk=\frac{1}{b-a}\int_a^bf(x)\mathrm{d}xk=b−a1​∫ab​f(x)dx, which lies within the minimum value mmm and maximum value MMM.

A more important property of substracting a constant number kkk from fff is, it does not change the value of ∫abf(x)g(x)dx\int_a^bf(x)g(x)\mathrm{d}x∫ab​f(x)g(x)dx as ∫abg(x)dx=0\int_a^bg(x)\mathrm{d}x=0∫ab​g(x)dx=0 is given. Then it follows that:

(∫abf(x)g(x)dx)2=(∫ab(f(x)−k)g(x)dx)2≤[∫abf(x)2dx−1b−a(∫abf(x)dx)2]∫abg(x)2dx\left(\int_a^bf(x)g(x)\mathrm{d}x\right)^2=\left(\int_a^b\left(f(x)-k\right)g(x)\mathrm{d}x\right)^2\le \left[\int_a^bf(x)^2\mathrm{d}x-\frac{1}{b-a}\left(\int_a^bf(x)\mathrm{d}x\right)^2\right]\int_a^bg(x)^2\mathrm{d}x (∫ab​f(x)g(x)dx)2=(∫ab​(f(x)−k)g(x)dx)2≤⎣⎢⎡​∫ab​f(x)2dx−b−a1​(∫ab​f(x)dx)2⎦⎥⎤​∫ab​g(x)2dx

The last inequality applies the standard C-S inequality.

Now all we need to do is to prove that:

∫abf(x)2dx−1b−a(∫abf(x)dx)2≤(M−mM+m)2∫abf(x)2dx\int_a^bf(x)^2\mathrm{d}x-\frac{1}{b-a}\left(\int_a^bf(x)\mathrm{d}x\right)^2\le \left(\frac{M-m}{M+m}\right)^2\int_a^bf(x)^2\mathrm{d}x ∫ab​f(x)2dx−b−a1​(∫ab​f(x)dx)2≤(M+mM−m​)2∫ab​f(x)2dx
